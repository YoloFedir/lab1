# Опис проєкту

Даний проєкт є програмою для шифрування та розшифрування зображень, зокрема для застосування у дронах. Програма оснащена графічним інтерфейсом на базі бібліотеки tkinter і надає можливість завантажити зображення, зашифрувати його за допомогою XOR-операції та зберегти отриманий результат.

Для коректного розшифрування необхідно використати той самий ключ seed, що й під час шифрування.

Використані технології

У проєкті задіяні такі технології:

	•	Python — основна мова реалізації;
	•	Tkinter — побудова графічного інтерфейсу користувача;
	•	Pillow — завантаження, обробка та збереження зображень;
	•	XOR-шифрування — алгоритм криптографічного перетворення;
	•	LFSR (лінійний регістр зсуву зі зворотним зв’язком) — генерація псевдовипадкової послідовності ключів.
